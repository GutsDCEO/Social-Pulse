## Getting Started

### Prerequisites

- Docker & Docker Compose
- Java 21+
- Maven 3.9+


### Run with Docker

```bash
docker compose up --build
```

### Run Locally

```bash
cd backend/auth-service
mvn spring-boot:run
```
